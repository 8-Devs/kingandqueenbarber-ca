# Logo notes

Original logo lives at: https://kingandqueenbarber.ca/storage/settings/logo.jpeg

This is a low-res JPEG (the worst possible format for a logo — JPEG compression artifacts around text, no transparency). In the rebuild we render the brand wordmark as live HTML/CSS instead, using a crown-mark + the King & Queen Barber wordmark in Playfair Display.

To add the original raster logo as a fallback later, download it manually and drop it here as `logo.jpeg`, then reference from `public/`.
